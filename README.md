# Anything-Landing-Page

A simple and modern landing page created for a clothing brand concept called **Anything.co**.
This project demonstrates basic **HTML and CSS layout skills**

## Overview

The landing page includes:

* A navigation header
* Hero section with brand introduction
* Featured clothing designs
* Quote section
* Call-to-action banner
* Footer

This project was built as part of practicing **front-end web development fundamentals**.
